# List All Traps in Online Instructions

Given the HTML for a page of online homework instructions, this script (`list-traps.py`) will list all the traps contained within that assignment in a nice, readable format.

## How it Works

How to use the script:

1. Go to the webpage with the online homework instructions and type Ctrl + S (or Cmd + S on Mac).
    - Drag the saved HTML file into this folder (see `./example_assignment_html.html` for example).
1. Run the script with the command `python ./list-traps.py ./<assignment page>.html`.
1. Use the output as you wish.

This script is **highly useful** when creating reports, because you can easily copy and paste all the traps into the report for the Honor Council to reference.
